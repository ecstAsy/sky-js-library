#### 方法集

##### Number
数学浮点运算（+、-、*、/）
- FloatTool

数字金额格式
- Numeral

##### Object
对象属性值Trim

#### 使用方法

npm安装依赖

> npm install sky-js-library

本地引入使用

```js
import { FloatTool } from 'sky-js-library';

FloatTool.add(2, 5);      // 7
FloatTool.subtract(5, 3); // 2
```
